# Week-1 Deliverables
Selected EV dataset from Kaggle
Defined problem statement:Predict the efficiency of an EV car based on its features.

The model is deployed using Streamlit for easy user interaction.

# Learning Type
Type	Description
Supervised Learning [Labeled dataset with target (EV efficiency)]
Regression [Predicts numerical output]

# Technologies Used

Python
Pandas, NumPy
Scikit-Learn
Streamlit
Pickle (for model saving)

# Project Files
File	Description
app.py	Streamlit Application
model.pkl	Trained Machine Learning Model
electric_vehicles_spec_2025.csv.csv	Dataset
README.md	Project Documentation

# How to Run the Application Locally
Step-1: Open Terminal / Command Prompt
Step-2: Run the command
python -m streamlit run app.py

Step-3: Press Enter
Streamlit will open automatically in your browser and you can use the EV prediction UI.
